esbuild v0.5.12にはコードの変更を検知して再ビルドする機能や、webpack-dev-serverのようなビルド後にブラウザを立ち上げてくれる機能がないので、[`sane`](https://github.com/amasad/sane)と[`reload`](https://github.com/alallier/reload)を使用してこれらの機能を実現しました。
